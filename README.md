# ALX Listing App

## Description
The ALX Listing App is an Airbnb clone project built using **Next.js**, **TypeScript**, and **TailwindCSS**. The goal of this project is to create a responsive and maintainable listing page where users can view property details in a clean and modern UI.

## Project Structure
- **components/**  
  Contains reusable UI components such as cards and buttons.
  
- **interfaces/**  
  Holds TypeScript interfaces for props and data models to ensure type safety.
  
- **constants/**  
  Stores reusable constants like API URLs, configuration settings, and static text.
  
- **public/assets/**  
  Contains images, SVGs, and other static assets used throughout the application.
